This intentionally insecure JavaScript Web Application was created by Björn Kimminich (@bkimminich) and is great for anyone coding or testing JavaScript that doesn’t understand all its security issues to watch out for. With both local and containerized environments available, Juice Shop is perfect for a fun challenge to offer in your organization.
